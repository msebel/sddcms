Class paging

Description
Description | Vars (details) | Methods (details)

Klasse die eine Verteilung von Records auf mehrere Seiten anbietet.

Die Seitenzahl wird seit sddCMS 2.0 in der Session gespeichert, sodass man nicht mehr für alle Links den 'page' Parameter führen muss.
