Class selectorControl

Description
Description | Vars (details) | Methods (details)

Erweiterung des BaseControl um einen Multiselektor.

Der Multiselektor muss nicht mit CTRL bedient werden. Er sendet für jeden Klick eine Anfrage per AJAX an den Server, welcher diese gleich speichern sollte.
