Wird verwendet zum darstellen und verarbeiten des Menus. Eine Instanz $Menu wird automatisch erstellt und ist im globalen Scope jederzeit verfügbar

		checkHierarchy (line 221)
	
 
	
	

Prüft ob das Menu Aufgrund der Hierarchie angezeigt wird.


Die Funktion geht davon aus, dass der Menupunkt nicht anzeigbar ist  Dies wird aber umgekehrt/ermöglicht wenn: 
  • Der gecheckte Menupunkt den gleichen Parent wie der aktuelle hat
  • Der gecheckte Menupunkt ein überobjekt des aktuellen ist
  • Der gecheckte Menupunkt in der ersten Hierarchie (0) ist

	
    
				
  • return: true = anzeigbar
