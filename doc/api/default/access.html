Class access

Description
Description | Vars (details) | Methods (details)

Diese Klasse regelt den Zugriff auf Seiten.

Sie verarbeitet ausserdem Logins von Benutzern / Administratoren. Es wird automatisch ein Objekt $Access als Instanz für dieses Objekt generiert. Dieses ist im gesamten globalen Scope vorhanden.
