Class displayObject

Bietet möglichkeiten ein File darzustellen.

Faktisch erstellt diese Klasse einfach eine Tabellenzelle in der verschiedene Linsk und Informationen zum File bereitgestellt werden. Bei Bildern erscheint ein Thumbnail.
